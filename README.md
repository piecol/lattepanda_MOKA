## MOKA - tiny carrier board for Lattepanda MU x86 SoM

Preview the KiCAD project on Kicanvas:
https://kicanvas.org/?github=https%3A%2F%2Fgithub.com%2Fpiecol%2Flattepanda_MOKA

What's on-board:

- Gigabit ethernet
- USB-PD @ 12V
- HDMI
- USB2.0 2X
- USB3.0
- M-Key M.2 PCIe 2230/42
- E-Key M.2 PCIe 2230/42
- CSI camera
- Debug UART
- QWIIK I2C
- FAN
